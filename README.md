# Висновки по Швидкостям Алгоритмів Пошуку Підрядка

## Стаття 1

1. **Алгоритм Боєра-Мура:**
   - Час виконання: 1.58e-05 секунд.
   - **Висновок:** Даний алгоритм має добру швидкодію при пошуку підрядка в статті 1.

2. **Алгоритм Кнута-Морріса-Прата:**
   - Час виконання: 1.60e-05 секунд.
   - **Висновок:** Швидкість виконання приблизно така ж, як і у Боєра-Мура.

3. **Алгоритм Рабіна-Карпа:**
   - Час виконання: 1.56e-05 секунд.
   - **Висновок:** Рабін-Карп показав подібну швидкодію, але трошки швидший за Боєра-Мура та Кнута-Морріса-Прата.

## Стаття 2

1. **Алгоритм Боєра-Мура:**
   - Час виконання: 1.56e-05 секунд.
   - **Висновок:** Швидкість алгоритму Боєра-Мура практично однакова для обох текстів.

2. **Алгоритм Кнута-Морріса-Прата:**
   - Час виконання: 1.56e-05 секунд.
   - **Висновок:** Кнут-Морріс-Пратт також демонструє стабільну швидкодію для обох текстів.

3. **Алгоритм Рабіна-Карпа:**
   - Час виконання: 1.54e-05 секунд.
   - **Висновок:** Рабін-Карп показав найшвидші результати серед усіх алгоритмів для обох текстів.

## Загальні Висновки

1. **Найшвидший алгоритм:**
   - Для обох текстів алгоритм Рабіна-Карп продемонстрував найкращі результати з точки зору часу виконання.

2. **Висновок:**
   - В залежності від конкретного застосування та особливостей даних, можна обрати відповідний алгоритм. У цьому випадку Рабін-Карп може бути бажаним вибором для ефективного пошуку підрядка.
